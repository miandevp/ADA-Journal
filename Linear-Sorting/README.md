## Counting Sort

### Idea

Contar cuántas veces aparece cada número y luego reconstruir el arreglo ordenado.

### Complejidad

- Tiempo: O(n + k)
- Espacio: O(k)

### Ejemplo

Arreglo original:

1 4 1 2 7 5 2

Arreglo ordenado:

1 1 2 2 4 5 7
